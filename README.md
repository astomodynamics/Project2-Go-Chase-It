## Build and launch
``` bash
$ mkdir catkin_ws && cd catkin_ws
$ git clone https://github.com/astomodynamics/Project2-Go-Chase-It.git
$ catkin_make
$ source devel/setup.bash
$ roslaunch my_robot world.launch

In another terminal
$ roslaunch ball_chaser ball_chaser.launch
```
